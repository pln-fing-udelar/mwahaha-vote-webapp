Info

This web page helps evaluate systems submitted by participants to the 2025-2026 MWAHAHA competition on Humor Generation. In this competition, participants submit computer program systems that are capable of generating humorous outputs given some context (e.g., a headline). This competition consists of two phases in which participants submitted systems: the Evaluation Trial phase (from October 2025 until Dec 15, 2025) and the Evaluation phase (Jan 10‚Äì31, 2026). We use this web page to evaluate the systems for both of these phases, one at a time (currently for the Evaluation phase).

Frequently Asked Questions (FAQ)

1. Am I allowed to vote/annotate?
Yes! Everybody's welcome!

2. Why would I vote/annotate these jokes?
To help us evaluate these systems and to have fun on the way (hopefully)!

3. What if neither option is funny?
If both jokes fail to make you laugh or are equally bad, please select "It's a tie ü§ù".

4. What if both are equally funny?
If both are genuinely funny, and you cannot decide which is better, you should also select "It's a tie ü§ù".

5. I don't understand the prompt/headline.
If you don't understand the prompt or headline, please click "Skip üö´" so we don't collect noisy data.

6. I don't understand a joke.
There's either a missing cultural context, or the system's output joke is nonsensical or gibberish. In the former case, it should be skipped (please, click "Skip üö´"). In the latter case, it should be considered as a failed joke attempt when deciding which vote to cast. In case of doubt, the safest is to skip to reduce the noise. Use your best judgment.

7. Why are some jokes very similar to each other?
We let the participants submit new versions of their systems. Different versions of the same system compete with each other, seamlessly with the rest of the systems. Chances are that they generate similar jokes.

8. Can I participate in this competition by submitting a system?
Unfortunately, the submission phase is now closed. Visit the 2025-2026 MWAHAHA competition website for more info. Note that, if participated in this competition (by submitting a system), you cannot vote/annotate on this web page.
